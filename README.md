# Cisco Network Projects

## Giới thiệu

Đây là bộ sưu tập các project mạng được xây dựng bằng **Cisco Packet Tracer** trong quá trình học tập và thực hành.

Mỗi project tập trung vào một hoặc nhiều chủ đề quan trọng trong quản trị mạng như VLAN, Routing, DHCP, DNS, ACL, NAT và Network Security.

Mục tiêu của repository là lưu trữ các bài lab, rèn luyện kỹ năng cấu hình thiết bị Cisco và xây dựng Portfolio phục vụ học tập cũng như ứng tuyển thực tập.

---

## Danh sách Project

### 01. Enterprise LAN
**Kiến thức:**
- VLAN
- Trunk
- Inter-VLAN Routing (Router-on-a-Stick)
- DHCP
- DNS
- Web Server

### 02. OSPF Multi Router
**Kiến thức:**
- Static Routing
- Default Route
- OSPF
- Routing Verification

### 03. Enterprise Network Security
**Kiến thức:**
- ACL (Access Control List)
- Port Security
- SSH
- Telnet
- Password Encryption
